# parthpandit182009.github.io

This repository now contains a React portfolio built with:

- Node.js
- React
- Tailwind CSS
- Framer Motion

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

## Notes

- The React app lives in `src/` with `public/index.html` as the app entry.
- Tailwind CSS and Framer Motion are already configured.
- Do not use Vite for this project.
